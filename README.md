Visit with Us: Wellness Package Prediction

This application predicts the likelihood of a customer purchasing the Wellness Tourism Package based on various demographic and interaction features.

Project Overview

Objective: Improve marketing efficiency by targeting high-probability customers.

Model: CatBoost/RandomForest (Trained via MLOps pipeline).

Deployment: Streamlit on Hugging Face Spaces.
